Add an Item

Item name is required.
Item name must be at least 4 characters long.
Item name cannot be longer than 30 characters.
Item name can only contains uppercase or lowercase letters or whitespace.
Item description is required.
Item description must be at least 4 characters long.
Item description cannot be longer than 30 characters.
Item description can only contain upper and lowercase letters, periods, commas, and question marks.
Item level is required.
Item level can only be 1 through 20.
Item class is required.
Item class can only be 1 through 10.
